# MetaWhisp — Бриф для создания лендинга и промо-видео

## Что это

MetaWhisp — нативное macOS приложение для мгновенной голосовой транскрипции. Работает полностью локально на Apple Silicon через Metal GPU (модель Whisper от OpenAI). Живёт в меню-баре, управляется глобальными хоткеями — нажал кнопку, сказал, текст вставился в любое приложение.

**Домен**: metawhisp.com (свободен, будет куплен)
**Платформа**: macOS 14+ (Apple Silicon)
**Цена**: пока бесплатно, планируется freemium

## Ключевые фичи (для лендинга)

### 1. Мгновенная транскрипция — 100% локально
- Нажимаешь правый ⌘ — говоришь — текст автоматически вставляется в текущее приложение
- Всё работает на устройстве через Metal GPU, никакие данные не уходят в облако
- 5 моделей Whisper на выбор: от Tiny (40 МБ, мгновенная) до Large V3 Turbo (950 МБ, максимальная точность)
- Поддержка 11 языков: русский, английский, испанский, французский, немецкий, китайский, японский, корейский, португальский, итальянский, украинский

### 2. Умная обработка текста — 3 режима
- **Raw**: дословная транскрипция без изменений
- **Clean**: автоматическое удаление слов-паразитов ("ну", "типа", "вот", "короче", "как бы", "это самое" и др.)
- **Polished**: ИИ-обработка через GPT-4o-mini — убирает паразиты, исправляет грамматику, структурирует в красивые предложения

### 3. Мгновенный перевод
- **Быстрый тап правый ⌥**: записать голос + сразу перевести
- **Долгое нажатие правый ⌥ (2 сек)**: перевести выделенный текст в любом приложении
- Автоопределение направления: кириллица → английский, латиница → русский
- Работает через OpenAI GPT-4o-mini

### 4. Самообучающийся словарь коррекций
- Приложение следит, исправляешь ли ты текст после вставки
- Запоминает исправления (особенно имена собственные, бренды, термины)
- В следующий раз автоматически применяет выученные коррекции
- Ручное управление словарём через UI

### 5. Подробная аналитика
- Дашборд с метриками: количество транскрипций, общее количество слов, время аудио, сэкономленное время
- Графики активности: по часам (сегодня), по дням (неделя/месяц), по месяцам (всё время)
- Анализ слов-паразитов: топ-10 с частотой
- Самые используемые слова (bar chart)
- Рекорды: самая длинная транскрипция, самая быстрая обработка
- Тренды: сравнение с предыдущим периодом (% изменения)

### 6. Меню-бар интеграция
- Иконка в меню-баре меняется в зависимости от состояния (волна → запись → обработка)
- Поповер с быстрым доступом: статус, последняя транскрипция, кнопка записи
- Плавающий оверлей-пилюля показывает состояние поверх всех окон
- Звуковая обратная связь (настраиваемая)

## Управление (хоткеи)

| Действие | Клавиша | Описание |
|----------|---------|----------|
| Запись/Стоп | Правый ⌘ | Начать/остановить запись из любого приложения |
| Запись + Перевод | Правый ⌥ (тап) | Записать и сразу перевести |
| Перевести выделенное | Правый ⌥ (удержание 2 сек) | Перевести выделенный текст |

## Технические характеристики (для секции "под капотом")

- **Движок**: WhisperKit 0.9.0 (Apple Silicon оптимизация)
- **GPU**: Metal acceleration для ML-инференса
- **Аудио**: 16 кГц, моно, PCM float32
- **Хранение**: SwiftData (SQLite) — всё локально
- **API**: OpenAI GPT-4o-mini (только для Polished режима и перевода)
- **Размер**: ~50 МБ приложение + 40-950 МБ на модель
- **RAM**: ~1-2 ГБ в покое
- **Code signing**: Hardened runtime с entitlements

## Для промо-видео: сценарии использования

### Сценарий 1: "Быстрая мысль"
Разработчик пишет код → нажимает правый ⌘ → диктует комментарий → текст автоматически вставляется в редактор → продолжает кодить. Всё за 3 секунды.

### Сценарий 2: "Митинг"
Человек на Zoom-звонке → нажимает правый ⌘ → записывает ключевые моменты → текст вставляется в Notion/заметки. Никто не видит, что он записывает.

### Сценарий 3: "Перевод на лету"
Читает статью на английском → выделяет абзац → удерживает правый ⌥ 2 секунды → абзац заменяется русским переводом прямо в текстовом поле.

### Сценарий 4: "Длинный текст"
Включает Polished режим → диктует 5 минут потока сознания → получает чистый, структурированный текст без единого "ну" и "короче".

### Сценарий 5: "Аналитика"
Открывает Dashboard → видит: "На этой неделе 47 транскрипций, 12,000 слов, сэкономлено 2 часа" → график показывает пик активности по вторникам.
